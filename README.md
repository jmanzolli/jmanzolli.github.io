# Jônatas A. Manzolli academic website

Static personal academic portfolio prepared for GitHub Pages.

## Files

- `index.html` - main one-page website
- `style.css` - responsive visual design
- `script.js` - active navigation highlighting
- `assets/Jonatas-Manzolli-CV.pdf` - downloadable CV

## Publish with GitHub Pages

1. Create a public GitHub repository named `jmanzolli.github.io` for a user site, or any repository name for a project site.
2. Upload these files to the repository root.
3. In GitHub, go to `Settings` -> `Pages`.
4. Choose `Deploy from a branch`, then select the `main` branch and `/root`.
5. The site will appear at `https://jmanzolli.github.io` for a user site, or `https://jmanzolli.github.io/repository-name/` for a project site.

## Content sources used

The content was compiled from the local CV PDF and public profiles: LinkedIn, Google Scholar, ResearchGate, Wix, ORCID, Sylff, CiênciaVitae, and GitHub.
